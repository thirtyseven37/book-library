## Description

Simple CRUD for book library. Used technologies:
* [TypeScript](https://github.com/microsoft/TypeScript)
* [Nest](https://github.com/nestjs/nest)
* [TypeORM](https://github.com/typeorm/typeorm)


## Installation

```bash
$ npm install
$ cp .env.example .env
```

Use your DB credentials in .env file and run migrations: 
```bash
$ npm run migrate-up
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
